How it works

  • Email Gateway (free, unlimited): Sends an email to your carrier's SMS-email address. Select your carrier and no API key is needed.
  • TextBelt (free tier): Uses the TextBelt public API — 1 free message per day, carrier detection automatic.

⚙️ Gateway method requires setting SMTP_USER and SMTP_PASS environment variables (Gmail App Password works great).
